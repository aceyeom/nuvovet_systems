## 3. Auto-Calculation Logic Observed

The EMR automatically resolves `mg/kg dose × patient weight → actual mg dose` inline per prescription line.

```
Example from screenshot:
Famotidine Inj.   →  0.5 mg/kg  ×  3.66 kg  =  1.83 mg
Furosemide Inj.   →  2.0 mg/kg  ×  3.66 kg  =  7.32 mg
```

**NuvoVet must replicate this:** display both the weight-based dose input AND the resolved actual dose per line item in the prescription table. Vets are accustomed to seeing both values simultaneously.

### 5.2 Prescription Line Item Schema (updated)

```javascript
{
  // ── Existing fields (keep) ──────────────────────
  id: string,
  name: string,
  drugId: string,
  dosePerKg: float,                 // mg/kg input
  unit: enum,                       // expanded unit enum

  folderName: string,               // 폴더명 grouping
```

### 5.3 Drug Database Entry Schema (updated)

```javascript
{
  // ── Existing fields (keep) ──────────────────────
  id, name, nameKr, activeSubstance, class, source,
  defaultDose, unit, freq, route,
  narrowTherapeuticIndex, cypProfile, riskFlags,
  renalElimination, pk, speciesNotes, contraindications,

  // ── New fields to add ──────────────────────────
  brandNames: string[],             // ["바이트릴", "Baytril"]
  dosageForm: enum,                 // "Tab" | "Inj" | "Cap" | "Susp" | "Drop" | "Oint"
  availableStrengths: [             // for dose picker
    { value: float, unit: string }
  ],
  washoutPeriod_days: integer,      // elimination half-lives × 5 rule
  speciesContraindicated: string[], // hardstop: ["cat"] or []

  organBurdenWeight: {
    renal: float,                   // 0–1 scale, derivable from renalElimination
    hepatic: float,                 // 0–1 scale
  },

  additiveRiskWith: {
    nephrotoxic: boolean,           // flags additive renal load
    hepatotoxic: boolean,
    gi_ulcer: boolean,              // NSAID + steroid class
    sedation: boolean,
    bleeding: boolean,
    qt_prolongation: boolean,
  },

  renalDoseAdjustment: {
    creatinine_threshold_dog: float,  // mg/dL trigger (typically 2.5–3.5)
    creatinine_threshold_cat: float,  // mg/dL trigger (typically 1.6–2.5)
    adjustment_type: enum,            // "reduce_dose" | "extend_interval" | "avoid"
    adjustment_factor: float,         // 0–1 multiplier (0.5 = 50% of normal dose)
    note: string,
  },

  hepaticDoseAdjustment: {
    applies: boolean,               // only true for narrowTherapeuticIndex + hepatic primary
    alt_threshold_multiplier: float, // e.g. 3.0 = ALT > 3× upper normal
    adjustment_type: enum,          // "reduce_dose" | "monitor" | "avoid"
    note: string,
  },

  contraindications: [              // restructured from string[] to object[]
    {
      condition: string,
      matchTerms: string[],         // for auto-matching patient conditions
      severity: enum,               // "absolute" | "relative" | "caution"
      action: enum,                 // "contraindicated" | "reduce_dose" | "monitor"
      labTrigger: {                 // optional — fires from patient lab values
        marker: string,             // "creatinine" | "alt" | "alp" | "bun"
        threshold: float,
        unit: string,
      }
    }
  ],
}
```

## 6. Priority Implementation Order

| Priority | Field Group | Reason |
|---|---|---|
| 🔴 P1 | `calculatedDose_mg` auto-calculation in RX table | Vets expect this — it's in every EMR |
| 🔴 P1 | `daysSupplied` + `timesPerDay` as numeric fields | Required for total dose and billing |
| 🔴 P1 | Structured `contraindications[]` with `matchTerms` | Powers automatic condition-matching engine |
| 🟡 P2 | `route` enum expansion to 8 values | Alignment with real clinical routes |
| 🟡 P2 | `additiveRiskWith` flags per drug | Polypharmacy detection beyond Plumb's DDI list |
| 🟡 P2 | `organBurdenWeight` per drug | Drives organ burden visualization accurately |
| 🟡 P2 | `renalDoseAdjustment` structured object | Patient-specific dosing — key differentiator |
| 🟢 P3 | Patient registration fields (insurance, vet IDs, etc.) | Required for EMR integration but not DUR engine |
| 🟢 P3 | `brandNames` + `dosageForm` + `availableStrengths` | Needed for drug search UX at scale |
| 🟢 P3 | `hepaticDoseAdjustment` | Limited vet literature — lower confidence value |
