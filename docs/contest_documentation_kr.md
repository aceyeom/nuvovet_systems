# 3.1 창업아이템의 개발 방안 / 진행(준비) 정도

## NuvoVet — 수의사를 위한 약물 안전성 검토 시스템

### 개발 배경 및 아이템 개요

NuvoVet은 수의사가 복합 약물 처방 시 발생할 수 있는 약물 상호작용, 장기 독성, 종 특이적 위험을 실시간으로 검토할 수 있도록 설계된 DUR(Drug Utilization Review) 시스템이다. 국내 동물병원에서는 다약제 처방이 일반화되어 있으나, 수의사 전용 의약품 안전성 검토 도구는 사실상 부재한 실정이다. 이에 Plumb's Veterinary Drug Handbook 기반의 구조화된 데이터와 규칙 기반 DUR 엔진을 결합하여, 임상 현장에서 즉시 활용 가능한 약물 안전성 검토 플랫폼을 개발하고 있다.

### 현재 개발 단계

프론트엔드(React/Vite)와 백엔드 데이터 구축이 완료된 단계이며, FastAPI 기반의 API 연동 작업이 진행 중이다. 핵심 데이터베이스는 877개 약물을 JSONL 형식으로 구조화하여 구축 완료하였다. 각 약물 레코드는 약물 정체성, 대사 및 청소, 장기 부하 점수, 용량 및 동태학, 약물 상호작용, 금기사항, 유전적 감수성 등 20개 이상의 섹션으로 구성되어 있으며, 개·고양이를 포함한 다종 동물에 대한 종별 데이터를 개별적으로 수록하고 있다.

### DUR 엔진 구성

현재 구현된 DUR 엔진은 8가지 핵심 규칙을 기반으로 동작한다.

1. **DDI 검출**: 약물 간 상호작용을 중증(Critical) / 중등도(Moderate) / 경증(Minor) 3단계 심각도로 분류하여 실시간 경고를 제공한다.
2. **장기 부하 점수**: 처방 내 전체 약물의 신장 및 간 누적 부담을 정량적으로 산출하고, 임계치 초과 시 경고를 표시한다.
3. **종 및 품종 안전성**: 고양이 독성 금기 약물, MDR1 유전자 변이 감수성 품종에 대한 하드스탑 및 감량 권고를 제공한다.
4. **체중 기반 용량 계산**: 환자 체중을 반영한 실제 투여량 산출 및 최대 용량 초과 여부를 실시간으로 확인한다.
5. **휴약 기간(Washout) 관리**: 세로토닌 증후군 위험을 포함한 약물 전환 시 안전 휴약 기간 가이드를 제공한다.
6. **CYP 효소 상호작용 탐지**: 기질-억제제-유도제 조합에 의한 대사 상호작용을 감지한다.
7. **QT 연장 누적 위험**: QT 연장 위험 약물의 중복 투여 시 심장 독성 경고를 발령한다.
8. **출혈 위험 중복 탐지**: 항응고제, NSAID, 혈소판 억제제의 복합 처방에 의한 출혈 위험 누적을 감지한다.

### 데이터 파이프라인 및 커버리지

약물 데이터는 Plumb's Veterinary Handbook을 1차 출처로 하여 구조화 추출 파이프라인을 통해 JSONL 포맷으로 변환하였다. 현재 국내 동물 허가 의약품(kr_vet), 오프라벨 사용 약물(off_label), 외국 수입 약물(foreign), 성분 불명 약물(unknown) 4개 카테고리를 커버하며, 각 약물에는 데이터 신뢰도 점수(`_data_quality.overall_confidence`)가 부여되어 있어 근거 수준을 처방의가 확인할 수 있다.

### 향후 계획

단기적으로는 FastAPI 기반 RESTful API 연동을 완료하고 국내 동물병원 2~3개소를 대상으로 임상 파일럿을 진행할 계획이다. 중기적으로는 국내 동물 EMR(전자의무기록) 시스템과의 연동을 통해 처방 데이터를 자동으로 수집하고, DUR 결과를 EMR 내에서 직접 확인할 수 있는 통합 환경을 구축할 예정이다. 장기적으로는 축적된 처방 데이터를 기반으로 한국 수의임상 환경에 최적화된 약물 안전성 데이터베이스를 지속적으로 고도화하고, 농림축산식품부 및 관련 학회와의 협력을 통해 국내 수의 DUR 표준 제정에 기여하는 것을 목표로 한다.
